对第四维度空间及高维空间中的金字塔的探索


此文章不适宜以下人群观看：

  1. 认为第四个维度是时间维度（即认为空间维度只有三个的人）；
  2. 无探索精神的人。

如下图，我们已经知道杨辉三角，如下图：

我们知道，这个表表示的是 (a+b) n 的各项系数，比如

(a+b) 4 = 1 a 4 + 4 a 3 b + 6 a 2 b 2 + 4 a b 3 + 1 b 4

但是，我们在中学阶段只接触到 (a+b) n ，换句话说，只有两个变量，a，和 b。

因为只有两个变量，所以我们的“杨辉三角”被限制在一个平面，也就是说，括号内的变量的数量决定了这个三角的维度。

所以，如果我们想在高维度继续我们的工作，把杨辉三角拓展到三维，甚至拓展到任意维度的空间，就必须增加括号中的变量的个数。

首先我们计算 (a+b+c) n 。

(a+b+c) 2 = 1 a 2 + 1 b 2 + 1 c 2 + 2 a b + 2 a c + 2 b c

(a+b+c) 3 = 1 a 3 + 1 b 3 + 1 c 3 + 3 a 2 b + 3 a 2 c + 3 b 2 a + 3 b 2 c + 3 c 2 a + 3 c 2 b + 6 a b c

(a+b+c) 4 = 1 a 4 + 1 b 4 + 1 c 4 + 4 a 3 b + 4 a 3 c + 4 b 3 a + 4 b 3 c + 4 c 3 a + 4 c 3 b +
            6 a 2 b 2 + 6 a 2 c 2 + 6 b 2 c 2 + 12 a 2 b c + 12 a b 2 c + 12 a b c 2
